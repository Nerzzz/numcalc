# Numcalc
Numcalc é uma calculadora web que converte números entre diferentes bases numéricas de forma instantânea.

## Demo
https://numcalcsd.vercel.app

## Funcionalidades
- Conversão de bases (binário, octal, decimal e hexadecimal);
- Escolhas personalizadas de entrada/saída;
- Resultado em tempo real.

## Tecnologias
- HTML;
- CSS;
- JavaScript (Vanilla).
